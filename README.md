# favourite_places

Favourite Places app allows users to add and name locations, displaying each place along with its corresponding location on a map for easy reference.

## Third party packages Used
* google_fonts: ^6.2.1
* uuid: ^4.4.0
* flutter_riverpod: ^2.5.1
* image_picker: ^1.1.1
* location: ^6.0.2
